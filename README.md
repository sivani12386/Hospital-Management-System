# 🏥 Hospital Management System

## 📌 Description
The Hospital Management System is a web-based application designed to manage hospital operations such as patient records, doctor details, appointments, and billing. It helps in organizing data efficiently, reducing manual work, and improving accuracy in hospital management.

---

## 🚀 Features
- Manage doctor information  
- Maintain patient records  
- Schedule appointments  
- Billing and payment management  
- Update and delete records  
- Easy search and data retrieval  

---

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: Java  
- Database: MySQL / Oracle (DBMS)

## ⚙️ How to Run

### 1. Setup Database
- Install MySQL / Oracle
- Create a database
- Import schema.sql

### 2. Run Backend

javac *.java
java Main


### 3. Run Frontend
- Open index.html in browser
